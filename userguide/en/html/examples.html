Appendix B. Examples

B.1. Bean

Create an instance of MyBean using its default constructor.


<deployment>
  <bean name="MyBean"
        interface="com.mypackage.MyInterface"
        class="com.mypackage.MyBean"/>
</deployment>

B.2. Constructor

Create an instance of MyBean using its String constructor.


<deployment>
  <bean name="MyBean"
        class="com.mypackage.MyBean">
    <constructor>
      <parameter>MyString</parameter>
    </constructor>
  </bean>
</deployment>

Create an instance of MyBean selecting its String constructor.


<deployment>
  <bean name="MyBean"
        class="com.mypackage.MyBean">
    <constructor>
      <parameter class="java.lang.String">MyString</parameter>
    </constructor>
  </bean>
</deployment>

Create an instance of MyBean selecting its String constructor and passing in null.


<deployment>
  <bean name="MyBean"
        class="com.mypackage.MyBean">
    <constructor>
      <parameter class="java.lang.String"><null/></parameter>
    </constructor>
  </bean>
</deployment>

Create an instance of MyBean using a static method on a factory passing in a String.


<deployment>
  <bean name="MyBean"
        interface="com.mypackage.MyInterface">
    <constructor factoryClass="com.mypackage.MyBeanFactory"
                 factoryMethod="create">
      <parameter class="java.lang.String">MyString</parameter>
    </constructor>
  </bean>
</deployment>

Create an instance of MyBean using a method on a factory instance passing in a String.


<deployment>
  <bean name="MyBeanFactory"
        class="com.mypackage.MyBeanFactory"/>

  <bean name="MyBean"
        interface="com.mypackage.MyInterface">
    <constructor factoryMethod="create">
      <parameter class="java.lang.String">MyString</parameter>
      <factory bean="MyBeanFactory"/>
    </constructor>
  </bean>
</deployment>

Injecting MyConstructorBean into the constructor.


<deployment>
  <bean name="MyConstructorBean"
        class="com.mypackage.MyConstructorBean">

  <bean name="MyBean"
        class="com.mypackage.MyBean">
    <constructor>
      <parameter><inject bean="MyConstructorBean"/></parameter>
    </constructor>
  </bean>
</deployment>

B.3. Property

Setting properties on MyBean


<deployment>
  <bean name="MyBean"
        class="com.mypackage.MyBean">
    <property name="MyStringProperty">MyString</property>
    <property name="MyDoubleProperty">0.0</property>
    <property name="MyIntProperty">42</property>
    <property name="MyBooleanProperty">true</property>
  </bean>
</deployment>

Setting a list property on MyBean


<deployment>
  <bean name="MyBean"
        class="com.mypackage.MyBean">
    <property name="MyListProperty">
      <list class="java.util.ArrayList"
            elementClass="java.lang.String">
        <value>MyStringValue1</value>
        <value>MyStringValue2</value>
        <value>MyStringValue3</value>
      </list>
    </property>
  </bean>
</deployment>

Setting a set property on MyBean


<deployment>
  <bean name="MyBean"
        class="com.mypackage.MyBean">
    <property name="MySetProperty">
      <set class="java.util.HashSet"
           elementClass="java.lang.String">
        <value>MyStringValue1</value>
        <value>MyStringValue2</value>
        <value>MyStringValue3</value>
      </set>
    </property>
  </bean>
</deployment>

Setting a map property on MyBean


<deployment>
  <bean name="MyBean"
        class="com.mypackage.MyBean">
    <property name="MyMapProperty">
      <map class="java.util.HashMap"
           keyClass="java.lang.String"
           valueClass="java.lang.String">
        <entry>
          <key>MyKey1</key>
          <value>MyString1</value>
        </entry>
        <entry>
          <key>MyKey2</key>
          <value>MyString2</value>
        </entry>
      </map>
    </property>
  </bean>
</deployment>

Injecting MyPropertyBean on MyBean


<deployment>
  <bean name="MyPropertyBean"
        class="com.mypackage.MyPropertyBean">

  <bean name="MyBean"
        class="com.mypackage.MyBean">
    <property name="MyProperty"><inject bean="MyPropertyBean"/></property>
  </bean>
</deployment>

Injecting MyPropertyBean's Property on MyBean


<deployment>
  <bean name="MyPropertyBean"
        class="com.mypackage.MyPropertyBean">

  <bean name="MyBean"
        class="com.mypackage.MyBean">
    <property name="MyProperty"><inject bean="MyPropertyBean" property="Property"/></property>
  </bean>
</deployment>

B.4. Depends

Create an explicit dependency on MyDependency.


<deployment>
  <bean name="MyDependency"
        class="com.mypackage.MyDependency"/>

  <bean name="MyBean"
        class="com.mypackage.MyBean">
    <depends>MyDependency</depends>
  </bean>
</deployment>

B.5. Install / Uninstall

Specify an install and an uninstall method for MyBean.


<deployment>
  <bean name="MyBean"
        class="com.mypackage.MyBean">
    <install method="myInstallMethod"/>
    <uninstall method="myUninstallMethod"/>
  </bean>
</deployment>

B.6. IgnoreCreate / IgnoreStart / IgnoreStop / IgnoreDestroy

Ignore the create, start, stop and destroy methods on MyBean.


<deployment>
  <bean name="MyBean"
        class="com.mypackage.MyBean">
    <ignoreCreate/>
    <ignoreStart/>
    <ignoreStop/>
    <ignoreDestroy/>
  </bean>
</deployment>
